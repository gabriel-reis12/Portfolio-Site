# Portfolio Site

Bem-vindo ao meu projeto de Portfólio!

Este site foi desenvolvido para apresentar quem eu sou, juntamente com minhas habilidades, experiências e projetos recentes. Ele serve como um currículo online dinâmico e interativo.

## O que você encontrará aqui:
- **Sobre Mim:** Um resumo sobre minha trajetória e paixão por tecnologia.
- **Habilidades:** As principais ferramentas e linguagens com as quais trabalho.
- **Projetos:** Uma seleção dos meus melhores trabalhos e o que desenvolvi neles.
- **Contato:** Informações de como falar comigo para oportunidades ou colaborações.

Sinta-se à vontade para explorar e entrar em contato!
